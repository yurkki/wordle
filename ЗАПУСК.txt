ИНСТРУКЦИЯ ПО ЗАПУСКУ WORDLE

1. Убедитесь, что у вас установлена Java 17 или выше
2. Откройте командную строку в папке проекта
3. Выполните одну из команд:

   Вариант 1 (через Gradle):
   gradlew.bat bootRun

   Вариант 2 (через JAR):
   gradlew.bat build
   java -jar build/libs/wordle-0.0.1-SNAPSHOT.jar

   Вариант 3 (через batch файл):
   run.bat

4. Откройте браузер и перейдите по адресу:
   http://localhost:8080

5. Наслаждайтесь игрой!

ПРАВИЛА ИГРЫ:
- Угадайте слово из 5 букв за 6 попыток
- Зеленый цвет - буква на правильном месте
- Желтый цвет - буква есть в слове, но на другом месте
- Серый цвет - буквы нет в слове
- Используйте только существующие русские слова

Если возникли проблемы:
1. Проверьте, что порт 8080 свободен
2. Убедитесь, что Java установлена правильно
3. Попробуйте изменить порт в application.properties
